# Gender Gap Clock

🙎🏾⏰


## Setup instructions

1) Clone the repository:

```
git clone https://github.com/CodeForAfrica/GenderGapClock.git
cd GenderGapClock
```

2) Download dependencies:

```
npm install
```

3) Build:

```
gulp
```

4) Set up a server and watch for changes:

```
gulp serve
```

+ Safari 9 and 10.
+ Opera 43 and 44.

### Mobile browsers:

+ Chrome for Android 56 and 57.
